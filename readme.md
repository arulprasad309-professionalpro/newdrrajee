# Dr. Rajeswari Autism Parenting Masterclass Website

A static landing page website for parents to sign up for autism parenting masterclass sessions.

## Pricing
- ₹200 per session

## How to Deploy on GitHub Pages
1. Upload files to GitHub repo
2. Go to Settings → Pages
3. Select Branch: main → /root
4. Save, and your website will be live
